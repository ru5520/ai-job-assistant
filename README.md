# AI Job Assistant

A simple Python project that compares a resume with a job description, extracts keywords, and identifies matched and missing skills.

## Features
- Read resume text from `resume.txt`
- Read job description text from `jd.txt`
- Extract keywords
- Filter stop words
- Identify matched keywords
- Identify missing keywords
- Save results to `output.txt`

## Project Structure
ai_job_assistant/
- app.py
- resume.txt
- jd.txt
- output.txt
- README.md
